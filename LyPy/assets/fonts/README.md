# LyPy fonts

On **first launch**, LyPy automatically downloads **Nunito Sans** (Bold + Regular) from the Fontsource CDN into:

`%LOCALAPPDATA%\LyPy\fonts\`

That is the built-in “Spotify-like” lyrics font (OFL license). No manual step needed if you have internet the first time you run the app.

## Optional: real Spotify Mix

Spotify Mix cannot be redistributed in this repo. To use the exact Spotify font, copy your `.ttf` files into `%LOCALAPPDATA%\LyPy\fonts\` (e.g. from [SpotiFont](https://github.com/houssamouhra/SpotiFont) releases), then restart LyPy.
