# About this repository

**LyPy-Testing** is where **Linux and Windows** cross-platform work happens (MPRIS on Linux, shared packaging, etc.).

| Repository | Purpose |
|------------|---------|
| **LyPy-Testing** (this repo) | Active development for Linux + Windows |
| **LyPy-macOS** (sibling / separate GitHub repo) | Snapshot of the Windows WMTC app; baseline for a future macOS port |

The macOS repo is created locally at `~/Projects/LyPy-macOS` unless you chose a different path. Push it to GitHub when ready (see that repo’s setup notes).
